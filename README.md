# Digital Literacy Project

This repository contains my Digital Literacy project completed as part of the CSE0001 course at VIT Bhopal.

## Project Overview

The project focuses on developing essential digital skills, including creating digital content, building an online portfolio, using collaborative platforms, understanding email etiquette, and learning about cyber safety.

## Tasks Included

- Task 1: Digital Literacy Infographic
- Task 2: Digital Portfolio
- Task 3: Platforms and Collaboration
- Task 4: Email Etiquette
- Task 5: Cybercrime Awareness
